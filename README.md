\# TruthLens - Deepfake Detection System



\## Setup Instructions



1\. Create virtual environment:



python -m venv venv

venv\\Scripts\\activate





2\. Install dependencies:



pip install -r backend/requirements.txt





3\. Run server:



cd backend/app

uvicorn main:app --reload





4\. Visit:



http://localhost:8000
